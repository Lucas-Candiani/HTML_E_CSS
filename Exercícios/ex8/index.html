Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em Negrito usando o termo B

Usando a tag STRONG (essa é semântica), temos termo em destaque

Itálico / Ênfase

Nessa frase possuimos um termo em negrito (Não semântico) utilizando a tag I

Nesta frase, temos um termo em ênfase usando a tag EM (semântico)

Texto marcado

Podemos marcar um texto (grifador escolar) utilizando a tag mark
e também podemos mudar a cor de um mark utilizando css marcador de outra cor

Texto grande e pequeno

Podemos colocar um texto grande usando big (Mas não é recomendado, visto que aumentar uma palavra é forma, deixa pras CSS's)
e podemos colocar um texto pequeno (leia as letras pequenas do contrato) utilizando small

Texto excluído

Para colocar um texto excluido que vai ser lido mas não deve ser considerado, podemos utilizar a tag del

Texto inserido / sublinhado

Para colocar um texto inserido de forma semântica utiliza-se a tag ins
e para colocar um texto sublinado não semântico (não recomendado) utiliza-se a tag u

Texto subrescrito

Para, por exemplo equações como x20+30

Texto subscrito

Para, por exemplo a água H2O
